1 Open Source Projects Worth Mentioning

These are all up on github and licensed with BSD-3.

  1. OCaml bindings library for geoip2, provides geolocation information from a given IP address. Pretty stable.
  2. OCaml bindings library and a command line tool to use the new Google compression algorithm/library called Brotli. I wrote it in C, C++, OCaml.
  3. OCaml bindings library for libssh, provides implementation of the ssh protocol. This is not stable, I need to work on it.
  4. Objective-C/C/OCaml powered native Cocoa application for OS X, its a GUI for ssh sessions, uses libssh, its called Tallgeese
  5. A Hodgepodge of functions for OCaml, really a prototyping library, ideal for hackathons.
